## About

The classic 2048 puzzle game. Slide tiles to combine numbers and reach 2048!

## Features

- Smooth animations
- Score tracking
- Best score saved
- Responsive touch/keyboard controls

## Installation

### From App Store
Open the **App Store** in zOS and search for "2048".

### Via npm
```bash
npm install @zos-apps/2048
```

## Usage

- Use arrow keys or swipe to move tiles
- Combine same numbers to double them
- Try to reach 2048!
